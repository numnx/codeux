# Supported Platforms

This document outlines the definitive list of compatible environments, including operating systems, web browsers, and mobile platforms.

## Operating Systems

The desktop application requires the following minimum operating system versions:

* **Windows**: Windows 10 (21H2) or higher
* **macOS**: macOS 12.0 or higher
* **Linux**: Ubuntu 22.04 LTS or higher

## Web Browsers

The web application is tested and officially supported on the following modern browsers. We support the current stable version and the prior major versions as indicated below:

* **Google Chrome**: Chrome N-2 or newer
* **Mozilla Firefox**: Firefox N-2 or newer
* **Microsoft Edge**: Edge N-2 or newer
* **Apple Safari**: Safari N-1 or newer

## Mobile Support

* **Mobile Web**: Fully supported via mobile browsers (e.g., Safari on iOS 15.0 or higher, Chrome on Android 12.0 or higher).
* **Native Mobile Apps**: There are currently no native mobile applications available for iOS or Android. Users should access the platform via the mobile web experience.
