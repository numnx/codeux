# Glossary

Here you will find definitions for app-specific concepts to reduce confusion.

## Assignee
A connected worker (agent) or human operator assigned to execute a specific Task. They are responsible for completing the work.

## MCP
Model Context Protocol. The communication standard used by the server to empower LLMs with codebase management, agent session creation, and intelligent sprint orchestration.

## Project
A collaboration boundary within a Workspace. It organizes Tasks and serves as a space for interactions between human operators and agents.

## Task
The fundamental unit of execution within a Project. A Task represents a single goal or subtask to be completed by an Assignee.

## Watcher
A user or agent who is subscribed to updates on a Project or Task. They receive notifications about progress but are not primarily responsible for execution.

## Webhook
An automated callback mechanism. It is used to send real-time event notifications from the system to external services.

## Workspace
The highest level of the core data model hierarchy, acting as the billing and organizational boundary. It contains Projects and manages global resources like AI Provider Account Management.
