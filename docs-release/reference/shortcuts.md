# Keyboard Shortcuts

This cheat sheet provides a quick reference for navigating and operating efficiently using keyboard shortcuts.

::: info Operating System Differences
For Windows and Linux users, substitute <kbd>Cmd</kbd> with <kbd>Ctrl</kbd>, and <kbd>Option</kbd> with <kbd>Alt</kbd> unless otherwise specified.
:::

## Global

| Action | Shortcut |
| :--- | :--- |
| Open Command Palette | <kbd>Cmd</kbd> + <kbd>K</kbd> |
| Create New Task | <kbd>Cmd</kbd> + <kbd>N</kbd> |
| Toggle Sidebar | <kbd>Cmd</kbd> + <kbd>B</kbd> |
| Open Settings | <kbd>Cmd</kbd> + <kbd>,</kbd> |
| Search | <kbd>Cmd</kbd> + <kbd>F</kbd> |

## Navigation

| Action | Shortcut |
| :--- | :--- |
| Go to Dashboard | <kbd>G</kbd> then <kbd>D</kbd> |
| Go to Projects | <kbd>G</kbd> then <kbd>P</kbd> |
| Go to Tasks | <kbd>G</kbd> then <kbd>T</kbd> |
| Next Tab | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>→</kbd> |
| Previous Tab | <kbd>Cmd</kbd> + <kbd>Option</kbd> + <kbd>←</kbd> |
| Navigate Back | <kbd>Cmd</kbd> + <kbd>[</kbd> |
| Navigate Forward | <kbd>Cmd</kbd> + <kbd>]</kbd> |

## Task Editing

| Action | Shortcut |
| :--- | :--- |
| Save Task | <kbd>Cmd</kbd> + <kbd>S</kbd> |
| Assign to me | <kbd>I</kbd> |
| Change Status | <kbd>S</kbd> |
| Focus Description | <kbd>Cmd</kbd> + <kbd>D</kbd> |
| Add Comment | <kbd>Cmd</kbd> + <kbd>Enter</kbd> |
| Cancel Edit | <kbd>Esc</kbd> |
