# Login Troubleshooting

## Overview
Use this guide if you are unable to log in to the application. Find your issue in the table below for a quick fix.

## Common Login Issues

| Problem | Likely Cause | Fix |
|---|---|---|
| Link says "Invalid or Expired" | Expired magic link | Request a new link from the sign-in page. Use it within 15 minutes. |
| Cannot provide 2FA code | Lost 2FA device | Contact your workspace admin to reset your 2FA, or reach out to support. |
| Redirect loop or error on SSO | SSO misconfiguration | Clear your browser cache. Try again. If it fails, contact your IT admin. |

## Contact Support

If all else fails, please contact support. We need specific information to help you quickly.

1. **Email Support**
   Send an email to support with the exact information below.

   ```text
   Account Email: [Your Email]
   Workspace Name or URL: [Your Workspace]
   Error Message: [Exact Error Text]
   Time of Issue: [Date and Time]
   Browser: [e.g., Chrome, Safari]
   ```

## Expected Result
You will successfully resolve your login issue or have the correct payload submitted to our support team.
