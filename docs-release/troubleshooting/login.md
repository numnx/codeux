# Login Troubleshooting

## Before you start
Use this guide if you are unable to log in to the application. Find your issue in the table below for a quick fix.

| Problem | Likely Cause | Fix |
|---|---|---|
| Link says "Invalid or Expired" | Expired magic link | Request a new link from the sign-in page. Use it within 15 minutes. |
| Cannot provide 2FA code | Lost 2FA device | Use a recovery code with **Use Recovery Code** on the verification screen. If unavailable, contact your workspace admin to reset 2FA. |
| Redirect loop or error on SSO | SSO misconfiguration | Clear your browser cache and retry. If it fails, contact your IT admin to verify SSO settings. |

## Contact Support

If all else fails, please contact support. We need specific information to help you quickly.
Send the payload below:

```text
Account Email: [Your Email]
Workspace Name or URL: [Your Workspace]
Error Message: [Exact Error Text]
Time of Issue: [Date and Time]
Browser: [e.g., Chrome, Safari]
```

## Expected Result
You will successfully resolve your login issue or provide support with the exact data needed to diagnose it.
