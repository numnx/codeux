# Windows Installation Guide

This guide provides step-by-step instructions for installing the application on Windows devices.

## Prerequisites

- A computer running Windows 10 or Windows 11.
- Administrator privileges may be required to bypass security warnings.

## Step 1: Download the Installer

1. Navigate to the release page and download the latest `.exe` installer file.
2. Save the file to a known location, such as your `Downloads` folder.

## Step 2: Run the Installer and Bypass SmartScreen

When you run the downloaded `.exe` file, Windows may display a "Windows protected your PC" prompt. This is the Windows SmartScreen warning, which appears because the application is not yet digitally signed.

To safely bypass this warning and proceed with the installation:

1. Double-click the downloaded `.exe` file.
2. If the blue Windows SmartScreen warning appears, click on **More info** at the end of the text.
3. Click the **Run anyway** button that appears at the bottom of the window.

::: warning
Only bypass SmartScreen warnings for `.exe` files you have downloaded from our official release channels.
:::

## Step 3: Complete the Installation

1. Follow the on-screen instructions in the setup wizard.
2. Once the installation is complete, you can launch the application from the **Start menu**.

## Step 4: Check for Updates

The application is configured to receive updates automatically, but you can also check for them manually.

1. Locate the application icon in the **System Tray** (near the clock on your taskbar).
2. Right-click the application icon.
3. Select **Check for Updates** from the context menu.

::: info
If an update is available, it will be downloaded and installed in the background. You may need to restart the application for the changes to take effect.
:::

## Expected Result

You will have successfully installed the application on your Windows machine, the app will be accessible from the Start menu, and you will know how to check for updates via the System Tray.
