# Installation Overview

This document provides a clear menu of choices for deciding how to run or install the app.

## Available Methods

1. **Linux/macOS Desktop**
2. **Windows Desktop**
3. **Self-compiling**
4. **Webbrowser**

## Which should I choose?

| Method | Pros | Cons | Setup Guide |
|--------|------|------|-------------|
| **Linux/macOS Desktop** | Easy to install, native OS integration, local access. | Takes up local disk space. | [Linux/macOS Guide](./linux-macos.md) |
| **Windows Desktop** | Easy to install, native OS integration, local access. | Takes up local disk space. | [Windows Guide](./windows.md) |
| **Self-compiling** | Full control over the build process, customizable, local access. | Requires technical knowledge and setup time. | [Self-compiling Guide](./self-compiling.md) |
| **Webbrowser** | No installation required, accessible anywhere, cloud sync. | Dependent on internet connection. | [Webbrowser Guide](./webbrowser.md) |
