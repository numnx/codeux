# Frequently Asked Questions

**1. Do you have a free tier?**
Yes, we offer a free tier to help you get started with the platform. For an overview of the release documentation, check out our [Home Page](/index).

**2. How do I switch workspaces?**
You can switch workspaces easily from the dashboard. For detailed instructions on managing your workspaces, refer to our [Workspaces Guide](/guides/workspaces).

**3. Can I export my data?**
Yes, you can export your workspace data at any time from the administration settings. For more information, check the [Home Page](/index).

**4. How do I rename a workspace?**
Renaming a workspace requires owner permissions. For step-by-step instructions, visit our [Workspaces Guide](/guides/workspaces).

**5. How do I delete a workspace?**
Deleting a workspace is a permanent action and requires owner permissions. Explore the complete guide in our [Workspaces Guide](/guides/workspaces).

**6. Where can I find task templates?**
We provide a standard task template for contributing. Check the [Templates Overview](/contributing/templates) for detailed specifications.

**7. How do I fix connection timeouts?**
Connection timeouts often occur due to firewall configurations or network restrictions. Please see common solutions in our [Home Page](/index).

**8. Are there any best practices for workspaces?**
We recommend organizing workspaces by team or project for optimal collaboration. Read our comprehensive recommendations in the [Workspaces Guide](/guides/workspaces).

**9. When was the latest version released?**
We regularly release updates with new features and bug fixes. You can find all the latest changes and version history in our [Home Page](/index).

**10. How can I manage user permissions?**
User roles and permissions can be customized by workspace administrators. For instructions on access control, please read the [Home Page](/index).
