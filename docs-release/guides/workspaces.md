# Manage Workspaces

## Switching workspaces

### Before you start
* You must have an active account.
* You must belong to more than one workspace.

### Steps
1. Navigate to the dashboard.
2. Click the **Workspace Selector** in the top navigation bar.
3. Select the desired workspace from the list.

### Expected result
The dashboard reloads and displays data for the selected workspace.

## Renaming a workspace

### Before you start
* You must have owner permissions for the workspace.

### Steps
1. Navigate to **Settings** in the main sidebar.
2. Select the **Workspace** tab.
3. Enter the new name in the **Workspace Name** field.
4. Click **Save**.

### Expected result
The workspace name updates in the navigation bar and settings panel.

## Deleting a workspace

### Before you start
* You must have owner permissions for the workspace.

> **Warning:** Deleting a workspace is a destructive action that will result in permanent data loss. All associated data will be unrecoverable.

### Steps
1. Navigate to **Settings** in the main sidebar.
2. Select the **Workspace** tab.
3. Scroll to the bottom and click **Delete Workspace**.
4. Enter the workspace name in the confirmation dialog.
5. Click **Confirm**.

### Expected result
The workspace is permanently deleted and you are redirected to the default workspace.
