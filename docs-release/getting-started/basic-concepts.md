# Basic Concepts

Understanding the organizational hierarchy is essential for effectively structuring your work. The system is built around three core layers: Workspaces, Projects, and Tasks.

## Hierarchy Overview

*   **Workspaces:** The highest level of organization. A Workspace acts as the primary billing and administrative boundary. It represents a single company, department, or distinct organization.
*   **Projects:** The collaborative layer within a Workspace. A Project serves as the boundary for teamwork, grouping related efforts, milestones, and the people executing them.
*   **Tasks:** The fundamental unit of execution. A Task represents a specific, actionable item of work that belongs to a Project.

## Concept Comparison

| Concept | Primary Function | Boundary Type | Example |
| :--- | :--- | :--- | :--- |
