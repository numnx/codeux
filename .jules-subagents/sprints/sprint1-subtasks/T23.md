# Task Specification: T23 - Sprint 1

## Objective
Extract the complex, inline Docker bootstrap shell script from `CliWorkflowService` into a dedicated, testable builder module in `src/infrastructure/providers/cli/docker-bootstrap-builder.ts`.

## Files to Modify

## Technical Details & Research Findings
- **Context**: The `bootstrapScript` is a long, multiline bash script constructed via array joins in the middle of `runProviderInDocker`. It handles credential syncing, PATH setup, and fallback installation.
- **Builder Design**:
    - Implement a `DockerBootstrapBuilder` class or functional builder.
    - Break the script into composable sections (e.g., `header`, `credentialSync`, `npmConfig`, `claudeAuth`, `fallbackInstall`, `execution`).
    - Support dynamic configuration (e.g., adding specific fallback install commands).
- **Constraints**: Ensure the generated script is functionally identical to the current one and maintains `set -euo pipefail`.
